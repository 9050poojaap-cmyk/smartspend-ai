# 💰 SmartSpend AI – Expense Tracker & Budget Analyzer

SmartSpend AI is a full-stack MERN application that helps users **track expenses, manage budgets, and gain financial insights** through an intuitive dashboard.

🔗 **Live App:** https://smartspend-ai-lime.vercel.app

---

## 🚀 Features

* 🔐 User Authentication (JWT)
* 📊 Dashboard with Income, Expenses & Savings
* 💸 Transaction Management
* 📈 Charts & Insights (Pie, Line, Bar)
* 🎯 Budget Alerts
* 📄 Export Reports as PDF
* 💱 INR currency support (₹ UI, Rs. in PDF)

---

## 🛠️ Tech Stack

* **Frontend:** React.js, Vite, Chart.js, jsPDF
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Deployment:** Vercel (Frontend), Render (Backend)

---

## ⚙️ Run Locally

```bash
git clone https://github.com/9050poojaap-cmyk/smartspend-ai.git
cd smartspend-ai
```

Backend:

```bash
cd backend
npm install
npm start
```

Frontend:

```bash
cd frontend
npm install
npm run dev
```
